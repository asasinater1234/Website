## 📁 Project Structure

```
/
├── index.html       # HTML structure and content
├── style.css        # All styling, layout, and animations
└── script.js        # Scroll reveal and nav highlighting
```

---
## ✨ Features

- **Responsive design** — works on desktop, tablet, and mobile
- **Scroll reveal animations** — sections fade in as you scroll
- **Active nav highlighting** — nav links highlight based on current section
- **Rotating hero badge** — animated NUS/degree badge in the hero
- **Hover micro-interactions** — cards lift and borders glow on hover
- **Zero dependencies** — no frameworks, no build step, pure HTML/CSS/JS

---

## 🎨 Design

- **Theme:** Dark techno aesthetic with a CSS grid background
- **Fonts:** [Syne](https://fonts.google.com/specimen/Syne) (headings) + [Space Mono](https://fonts.google.com/specimen/Space+Mono) (labels/code) via Google Fonts
- **Colours:** `#0a0e1a` background · `#00e5ff` cyan accent · `#ff6b35` orange accent

To change the colour scheme, edit the CSS variables at the top of `style.css`:

```css
:root {
  --bg: #0a0e1a;
  --accent: #00e5ff;
  --accent2: #ff6b35;
  /* ... */
}
```
